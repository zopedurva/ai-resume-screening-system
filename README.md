# AI Resume Screening System

AI Resume Screening System built using Python, NLP, and Machine Learning to compare resumes with job descriptions and generate candidate match scores using TF-IDF and cosine similarity.

## Features
- User registration and login
- Resume upload in PDF format
- Job description input
- Resume-job match score calculation
- Matching keyword extraction
- Analysis history storage using SQLite

## Tech Stack
- Python
- Streamlit
- SQLite
- scikit-learn
- NLTK
- PyPDF2
